# Calculator — coeficientul de pat k_s (NP 112/2014)

Aplicație web **de sine stătătoare** (un singur fișier) pentru calculul coeficientului de pat
(modulul de reacțiune al terenului, *k_s*) după **NP 112/2014, Anexa K**.

## Cum se folosește
Deschideți **`index.html`** prin dublu-click în orice browser (Chrome, Edge, Firefox).
- Rulează **100% local**, fără internet și fără instalare.
- Poate fi copiat pe stick / trimis prin e-mail — funcționează oriunde.

## Ce conține
Cele patru metode din normativ de determinare a *k_s*:
1. **Proba de încărcare cu placa** (K.6, K.7) — corelare coeziv / necoeziv.
2. **Din parametrii de compresibilitate** — K.8 (E_s, ν_s, cu k_m interpolat din Tabelul K.1)
   și K.9 (E_oed).
3. **Valori orientative** — Tabelul K.2 (după I_D / I_C).
4. **Calcul invers** din tasare — K.10.

Plus:
- **Aplicare**: calculul coeficientului caracteristic λ și al lungimii elastice l_e pentru
  grinda continuă pe mediu Winkler (K.3.2.1).
- **Tabelele K.1 și K.2** afișate integral.
- Explicații, pași de calcul detaliați și avertismente (inclusiv nota de inconsistență
  dimensională a formulei K.8).
- Comutator temă luminoasă / întunecată (util la proiector).

## Important — material didactic
Calculatorul este **non-normativ**. Singura autoritate rămâne textul oficial NP 112/2014.
Valorile și formulele sunt preluate din Anexa K (ec. K.5–K.10, Tabelele K.1, K.2).
Folder separat, în afara vault-ului wiki — nu afectează baza de cunoștințe.
